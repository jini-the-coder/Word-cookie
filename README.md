## 📌 About

**Word Cookie** is a Python implementation of the classic word-search puzzle game. Players are presented with a jumble of letters and must find valid words hidden inside the grid. The game blends simple mechanics with surprisingly addictive gameplay — perfect for anyone who enjoys reading, vocabulary, and casual brain-teasers.

The project was originally built for the **VNPS'18 intercollege showcase**, hence the script name `vnps.py`.

## ✨ Features

- 🎯 **Find hidden words** within a jumble of letters

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

### Run the Game

```bash
# 1. Clone the repository
git clone https://github.com/jini-the-coder/Word-cookie.git
cd Word-cookie

# 2. Run the game
python vnps.py
```

That's it — no extra dependencies required.

## 🎮 How to Play

1. Launch the game with `python vnps.py`.
2. A jumble of letters appears on screen.
3. Identify valid words hidden in the jumble.
4. Submit your guesses — find as many words as you can!
